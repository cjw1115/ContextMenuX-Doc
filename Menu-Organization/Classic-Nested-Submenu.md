# Classic Nested Submenu

> **Experimental**

A toggle in Settings that enables editing of third-level (3rd-level) submenus for the **classic** context menu.

## What It Does

When enabled, you can create **3-level deep** submenu hierarchies in your menu items. These nested submenus are **only visible in the classic context menu** (the "Show more options" menu).

> **Note:** The Windows 11 modern context menu does not support displaying 3rd-level submenus. Items nested at this depth will only appear in the classic menu.

## When to Use

Turn this on if you need deeper menu organization and your workflow relies on the classic context menu.
