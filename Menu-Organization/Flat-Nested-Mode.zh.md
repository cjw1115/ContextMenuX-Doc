# 平铺 / 嵌套模式

控制菜单项在 Windows 11 现代右键菜单中的组织方式。

## 平铺模式

每个菜单项作为**独立的顶级条目**显示在右键菜单中。

```
右键菜单：
├── 用工具 A 打开
├── 转换为 PDF
└── 上传到服务器
```

适用于：数量较少、需要快速访问的菜单项。

## 嵌套模式

所有菜单项归入一个自定义名称的**单一父级条目**下。

```
右键菜单：
└── 我的工具 ▸
    ├── 用工具 A 打开
    ├── 转换为 PDF
    └── 上传到服务器
```

- 为父级条目输入一个**显示名称**（如"我的工具"）。
- **仅适用于 Windows 11 现代右键菜单。** 嵌套模式利用系统的自动折叠功能，经典菜单中不可用。

适用于：菜单项较多时保持右键菜单整洁。

## 子菜单容器

在任一模式下，你可以通过开启菜单项的**"包含子项"**来创建层级结构。这会将该项变为子菜单容器，容纳子项而非执行命令。
