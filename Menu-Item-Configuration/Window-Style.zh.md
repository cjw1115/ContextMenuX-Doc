# 窗口样式

控制启动的应用程序窗口的显示方式。

## 选项

| 样式 | 说明 |
|------|------|
| **正常** | 窗口以默认大小和位置显示（默认值） |
| **最小化** | 窗口启动时最小化到任务栏 |
| **最大化** | 窗口启动时最大化填满屏幕 |
| **隐藏** | 窗口以不可见方式启动——适用于后台脚本或静默命令 |

## JSON 配置

```json
{
    "WindowStyle": "Hidden"
}
```

接受：`Normal`、`Minimized`、`Maximized`、`Hidden`（不区分大小写）。如果省略，默认为 `Normal`。
