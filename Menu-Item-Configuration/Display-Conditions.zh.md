# 显示条件

显示条件允许你根据右键点击的文件或文件夹的属性，控制菜单项何时出现。

## 条件字段

每个条件匹配以下文件属性之一：

| 字段 | 变量 | 说明 | 示例 |
|------|------|------|------|
| 路径 | `[path]` | 点击项目的完整路径 | `C:\Users\me\Documents\report.pdf` |
| 名称 | `[name]` | 包含扩展名的文件名 | `report.pdf` |
| 无扩展名名称 | `[stem]` | 不含扩展名的文件名 | `report` |
| 扩展名 | `[ext]` | 文件扩展名（含点号） | `.pdf` |
| 父文件夹名称 | `[dirname]` | 父目录的名称 | `Documents` |
| 父文件夹 | `[dir]` | 父目录的完整路径 | `C:\Users\me\Documents` |

## 匹配模式

| 模式 | 说明 | 示例模式 | 匹配结果 |
|------|------|----------|----------|
| 精确匹配 | 不区分大小写的精确匹配 | `.pdf` | 仅匹配 `.pdf` |
| 包含 | 值包含该模式 | `report` | `report.pdf`、`annual_report.docx` |
| 开头匹配 | 值以该模式开头 | `IMG_` | `IMG_001.jpg`、`IMG_photo.png` |
| 结尾匹配 | 值以该模式结尾 | `_backup` | `data_backup`、`db_backup` |
| 通配符 | Glob 风格的 `*` 和 `?` 匹配 | `*.log` | `app.log`、`error.log` |
| 正则表达式 | 正则表达式（不区分大小写） | `^IMG_\d+$` | `IMG_001`、`IMG_9999` |

## 行为：显示条件 / 隐藏条件

每个条件为**显示条件**或**隐藏条件**。

- **显示条件** — 当此条件匹配时，菜单项应当出现。
- **隐藏条件** — 当此条件匹配时，菜单项应当隐藏。

## 多个条件如何组合

当定义了多个条件时，它们按以下逻辑组合：

1. **隐藏条件**优先以 **OR** 逻辑计算——如果**任何**隐藏规则匹配，菜单项立即隐藏。
2. **显示条件**以 **OR** 逻辑计算——如果**任何**显示规则匹配，菜单项通过显示检查。
3. 如果存在显示规则但**没有任何**匹配，菜单项将被隐藏。
4. 如果不存在显示规则，菜单项默认显示（仅隐藏规则生效）。

简而言之：**显示规则之间 OR 组合，隐藏规则之间 OR 组合，隐藏优先于显示。**

### 示例

#### 为多种文件类型显示

| 规则 | |
|------|-|
| 显示条件：扩展名等于 `.jpg` | |
| 显示条件：扩展名等于 `.png` | |
| 显示条件：扩展名等于 `.gif` | |

结果：菜单项对任何 `.jpg`、`.png` 或 `.gif` 文件显示。

#### 为某类型显示但排除特定名称

| 规则 | |
|------|-|
| 显示条件：扩展名等于 `.log` | |
| 隐藏条件：名称包含 `debug` | |

| 文件 | 结果 |
|------|------|
| `app.log` | 显示 |
| `debug.log` | 隐藏（隐藏规则匹配） |
| `report.pdf` | 隐藏（没有显示规则匹配） |

#### 仅隐藏规则（无显示规则）

| 规则 | |
|------|-|
| 隐藏条件：名称包含 `temp` | |
| 隐藏条件：扩展名等于 `.bak` | |

结果：菜单项对所有文件显示，**除了**名称包含 `temp` 或扩展名为 `.bak` 的文件。
