# 多选行为

控制在资源管理器中选中多个文件时菜单项的行为。

## 模式

| 模式 | 说明 |
|------|------|
| **为每个文件分别执行** | 对每个选中的文件分别执行命令 |
| **一次传递所有路径** | 在单次执行中传递所有选中的文件路径 |
| **仅在单选时显示** | 选中多个文件时隐藏菜单项 |

## 一次传递所有路径

使用此模式时：

- 在启动参数中使用 `[paths]` 或 `[paths:q]`（带引号）接收所有选中的路径。
- 指定**分隔符**来分隔路径（默认为空格）。
- 如果忘记添加 `[paths]` / `[paths:q]`，将只使用第一个文件的路径。

### 示例

**使用工具合并 PDF：**

```
Target:   C:\Tools\pdfmerge.exe
Args:     --output merged.pdf [paths:q]
```

选中 `a.pdf`、`b.pdf`、`c.pdf` 后执行：

```
C:\Tools\pdfmerge.exe --output merged.pdf "a.pdf" "b.pdf" "c.pdf"
```
