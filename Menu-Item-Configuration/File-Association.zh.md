# 文件关联

控制哪些文件类型和位置触发你的右键菜单项。

## 快捷选项

| 选项 | 效果 |
|------|------|
| **所有文件** | 匹配任何文件（`*`） |
| **文件夹** | 在文件夹上右键时显示 |
| **桌面** | 在桌面背景上右键时显示 |

## 自定义文件扩展名

添加特定扩展名（如 `.png`、`.txt`、`.log`）以将菜单项限制为这些文件类型。

- 输入以 `.` 开头的扩展名，按回车添加。
- 使用**从文件检测**自动检测已有文件的扩展名。
- 点击标签上的 `×` 移除扩展名。

## 验证

扩展名必须：
- 以句点（`.`）开头
- 不包含 `\ / : * ? " < > |`
- 不以空格或句点结尾
