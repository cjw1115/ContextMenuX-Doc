# Context Menu X Documentation

Context Menu X lets you create custom right-click context menu items for Windows 11 File Explorer.

## Menu Item Configuration

- [Launch Arguments](Menu-Item-Configuration/Launch-Arguments.md) — Set the target program and command-line arguments with path variables
- [Working Directory](Menu-Item-Configuration/Launch-Environment.md) — Set the working directory for the launched process
- [Window Style](Menu-Item-Configuration/Window-Style.md) — Control the window appearance (Normal, Minimized, Maximized, Hidden)
- [File Association](Menu-Item-Configuration/File-Association.md) — Choose which file types and locations trigger the menu item
- [Multi-Selection](Menu-Item-Configuration/Multi-Selection.md) — Control behavior when multiple files are selected
- [Display Conditions](Menu-Item-Configuration/Display-Conditions.md) — Show or hide menu items based on file/folder properties

## Menu Organization

- [Flat / Nested Mode](Menu-Organization/Flat-Nested-Mode.md) — Choose between top-level entries or a grouped submenu
- [Classic Nested Submenu](Menu-Organization/Classic-Nested-Submenu.md) — Enable nested submenus in the legacy context menu

## Settings

- [Signing Certificate](Settings/Certificate.md) — Generate or import certificates for MSIX package signing
