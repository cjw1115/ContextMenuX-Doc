# Context Menu X 文档

Context Menu X 允许你为 Windows 11 文件资源管理器创建自定义右键菜单项。

## 菜单项配置

- [启动参数](Menu-Item-Configuration/Launch-Arguments.zh.md) — 设置目标程序和命令行参数（含路径变量）
- [工作目录](Menu-Item-Configuration/Launch-Environment.zh.md) — 设置启动进程的工作目录
- [窗口样式](Menu-Item-Configuration/Window-Style.zh.md) — 控制窗口外观（正常、最小化、最大化、隐藏）
- [文件关联](Menu-Item-Configuration/File-Association.zh.md) — 选择哪些文件类型和位置触发菜单项
- [多选行为](Menu-Item-Configuration/Multi-Selection.zh.md) — 控制选中多个文件时的行为
- [显示条件](Menu-Item-Configuration/Display-Conditions.zh.md) — 根据文件/文件夹属性显示或隐藏菜单项

## 菜单组织

- [平铺 / 嵌套模式](Menu-Organization/Flat-Nested-Mode.zh.md) — 选择顶级条目或分组子菜单
- [经典嵌套子菜单](Menu-Organization/Classic-Nested-Submenu.zh.md) — 在旧版右键菜单中启用嵌套子菜单

## 设置

- [签名证书](Settings/Certificate.zh.md) — 生成或导入用于 MSIX 包签名的证书
